
🎯 Number Guessing Game (Python)

A simple command-line based Number Guessing Game built using Python.
The computer randomly selects a number between 1 and 100, and the player has 10 attempts to guess it correctly.

🚀 How the Game Works

The system generates a random number between 1 and 100.

The player gets 10 chances to guess the number.

After each wrong guess:

If the guess is too high → it shows "lesser"

If the guess is too low → it shows "higher"

If the player guesses correctly → 🎉 "congrats! you won"

If all attempts are used → ❌ "oh! attempts over"

🧠 Concepts Used

random module

for loop

if-else conditionals

User input handling

Basic game logic implementation
